# Salla Twilight Starter Theme

هذا قالب بداية مبني ليتوافق مع استيراد Salla عبر `twilight.json` وبنية ملفات Twilight الأساسية.

## الملفات المهمة
- `twilight.json`
- `src/views/layouts/master.twig`
- `src/views/pages/index.twig`
- `src/views/pages/product/single.twig`

## الخطوات
1. فك الضغط.
2. استبدل محتوى مستودع GitHub الحالي بهذه الملفات.
3. ارفعها إلى نفس الريبو العام.
4. من بوابة شركاء سلة اختر **استيراد من GitHub**.

## ملاحظات
- هذا **Starter Theme** وليس نسخة نهائية من ثيم متجر مكتمل.
- بعد الاستيراد ستحتاج غالبًا تخصيصات إضافية وربطًا أعمق مع مكونات Twilight الجاهزة.
- السبب: لا يمكنني اختبار الاستيراد مباشرة داخل حسابك من هنا.
